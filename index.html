Builds

LEO pass planner and link margin

Input a TLE and a ground site. Get pass times and a margin timeline using FSPL with ITU gaseous and rain loss options. Badges show frame TEME and TLE epoch.

Status: in progress. Stack: FastAPI, sgp4, astropy, React.

  • Endpoints: /passes, /margin, /export/ics
  • Target accuracy: rise, max, set within 2 s vs reference
  • Exports: ICS, CSV, PDF summary

Latency under load probe

Measures latency, jitter, loss, and short outages at set offered loads. Works with Starlink or any ISP. Produces charts and CSV.

Status: in progress. Stack: Python agent, FastAPI, React.

  • Stages: idle, 10 percent, 50 percent, 80 percent
  • Detections: micro outages of at least 1 s, availability
  • Outputs: dashboard, CSV export

Local first. Then deploy. No client secrets.
