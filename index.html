Work (job-first)

LEO Pass Planner + Link-Margin Timeline

Given a TLE and ground site, compute passes and plot margin vs time using transparent FSPL + ITU gaseous/rain loss. Always shows frame=TEME and TLE epoch.

Status: Building | Tech: FastAPI, sgp4, astropy/ERFA, React+Chart.js

  • API: /passes, /margin, /export/ics
  • Accuracy target: rise/max/set within ±2 s vs reference
  • Exports: ICS, CSV, PDF one-pager

Latency-Under-Load Probe

Measures latency, jitter, loss, and micro-outages at 10/50/80% offered load. Useful for Starlink/any ISP. Reproducible methods; CSV export.

Status: Building | Tech: Python agent + FastAPI + React dashboard

  • Stages: idle → 10% → 50% → 80% uplink
  • Detections: micro-outages ≥1 s; availability%
  • Outputs: charts + raw CSV

Both projects are engineered to be demoable and testable locally, then deployed. No client-side secrets; tokens live server-side.
